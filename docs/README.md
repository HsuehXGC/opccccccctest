## 实现状态说明

文档描述的是**系统设计**。当前代码状态：

- **已实现（M1）**：完整前台（Vite + React），由浏览器内 mock 数据 + 模拟执行引擎驱动；后端 Agent 网关与协议骨架
- **规划中（M2/M3）**：真实 agent 接入、真实执行闭环、数据库持久化与账号鉴权

文中凡涉及"规划/目标/M2/M3"的部分为设计蓝图，尚未落地实现。
